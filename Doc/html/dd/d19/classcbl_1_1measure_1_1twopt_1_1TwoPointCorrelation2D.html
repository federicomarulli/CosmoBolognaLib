Detailed Description

The class TwoPointCorrelation2D.

"Headers/TwoPointCorrelation2D.h"

This class is used to handle objects of type TwoPointCorrelation2D . It is used to measure the 2D two-point correlation function, in Cartesian and polar coordinates, \(\xi(r_p,\pi)\) or \(\xi(r,\mu)\), that is as a function of perpendicular, \(r_p\), and parallel, \(\pi\), line-of-sight separations, and as a function of absolute separation, \(r=\sqrt{r_p^2+\pi^2}\), and the cosine of the angle between the separation vector and the line of sight, \(\mu\equiv\cos\theta=s_\parallel/s\), respectively.

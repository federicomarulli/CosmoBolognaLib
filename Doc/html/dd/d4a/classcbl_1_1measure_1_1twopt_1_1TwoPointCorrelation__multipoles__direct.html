Detailed Description

The class TwoPointCorrelation_multipoles_direct.

TwoPointCorrelation_multipoles_direct.h "Headers/TwoPointCorrelation_multipoles_direct.h"

This class is used to handle objects of type TwoPointCorrelation_multipoles_direct . It is used to measure the first three non-null multipole moments of the two-point correlation function, i.e. the monopole, the quadrupole and the exadecupole \(\xi_l(r) = (2l+1) \int_{\mu_{min}}^{\mu_{max}} \xi(s,\mu) L_l(\mu) d\mu\), with \(l=0,2,4\).

This class uses the so-called direct method: each pair is weighted with the Legendre polynomial of the corresponding multipole, computed at the cosine of the angular separation between the objects

The monopole estimated with the direct method can be obtained with the class TwoPointCorrelation_monopole

Examples
2pt_multipoles.cpp, and model_2pt_multipoles.cpp.

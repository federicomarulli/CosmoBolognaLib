The class TwoPointCorrelation2D_cartesian.

TwoPointCorrelation2D_cartesian.h "Headers/TwoPointCorrelation2D_cartesian.h"

This class is used to handle objects of type TwoPointCorrelation2D_cartesian . It is used to measure the 2D two-point correlation function in cartesian coordinates, \(\xi(r_p,\pi)\), that is as a function of perpendicular, \(r_p\), and parallel, \(\pi\), line-of-sight separations.

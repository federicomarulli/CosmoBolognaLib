Detailed Description

The class TwoPointCorrelation1D_angular.

TwoPointCorrelation1D_angular.h "Headers/TwoPointCorrelation1D_angular.h"

This class is used to handle objects of type TwoPointCorrelation1D_angular . It is used to measure the angular two-point correlation function, \(w(\theta)\), defined as \(dP_{12}=n^2[1+w(\theta)]dV_1dV_2\), where \(n\) is the average number density, and \(dP_{12}\) is the probability of finding a pair with one object in the volume \(dV_1\) and the other one in the volume \(dV_2\), separated by an angle \(\theta\).

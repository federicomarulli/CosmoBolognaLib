set the dispersion model to fit the wedges of the two-point correlation function

the wedges of the two-point correlation function are computed by cbl::modelling::twopt::xiWedges, in which the redshift-space matter power spectrum \(P(k, \mu)\) is modelled with the so-called dispersion model, computed by cbl:modelling::twopt::Pkmu_dispersion (see e.g. Pezzotta et al. 2017, https://arxiv.org/abs/1612.05645)

the model has 5 parameters:

  • \(f(z)\sigma_8(z)\)
  • \(b(z)\sigma_8(z)\)
  • \(\sigma_v\)
  • \(\alpha_{\perp}\)
  • \(\alpha_{\parallel}\)

the dark matter two-point correlation function is computed using the input cosmological parameters

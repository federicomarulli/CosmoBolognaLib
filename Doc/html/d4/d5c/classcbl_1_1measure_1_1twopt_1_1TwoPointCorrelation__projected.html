Detailed Description

The class TwoPointCorrelation_projected.

TwoPointCorrelation_projected.h "Headers/TwoPointCorrelation_projected.h"

This class is used to handle objects of type TwoPointCorrelation_projected . It is used to measure the projected two-point correlation function, \(w(r_p)=2\int^{\pi_{\rm max}}_{0} d\pi' \, \xi(r_p, \pi')\).

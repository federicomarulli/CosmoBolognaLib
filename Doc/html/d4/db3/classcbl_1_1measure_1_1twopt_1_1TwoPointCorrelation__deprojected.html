Detailed Description

The class TwoPointCorrelation_deprojected.

"Headers/TwoPointCorrelation_deprojected.h"

This class is used to handle objects of type TwoPointCorrelation_deprojected . It is used to measure the deprojected two-point correlation function, \(\xi(r)=-\frac{1}{\pi}\int^{r_{\rm max}}_r dr_p' \frac{dw_p(r_p')/dr_p}{\sqrt{r_p'^2-r^2}}\).

The class TwoPointCorrelation_wedges.

"Headers/TwoPointCorrelation_wedges.h"

This class is used to handle objects of type TwoPointCorrelation_wedges . It is used to measure the wedges \(\xi_\perp, \xi_\parallel\) as \(\xi_{\Delta\mu} = \frac{1}{2} \int_{\mu_0}^{\mu_1} \xi(s,\mu)d\mu\) with \(\mu_0=0\) and \(\mu_1=0.5\) for \(\xi_\perp\) and \(\mu_0=0.5\) and \(\mu_1=1\) for \(\xi_\parallel\)

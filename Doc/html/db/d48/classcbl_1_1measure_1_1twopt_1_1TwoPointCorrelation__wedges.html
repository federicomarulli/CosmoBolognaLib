Detailed Description

The class TwoPointCorrelation_wedges.

"Headers/TwoPointCorrelation_wedges.h"

This class is used to handle objects of type TwoPointCorrelation_wedges . It is used to measure the wedges of the two-point correlation function, \(\xi_{\Delta\mu} = \frac{1}{\Delta\mu} \int_{\mu_0}^{\mu_1} \xi(s,\mu)d\mu\). The default case is the one with two wedges, perpendicular - \(\xi_\perp\) and parallel - \(\xi_\parallel\), where \(\mu_0=0\) and \(\mu_1=0.5\) for \(\xi_\perp\), and \(\mu_0=0.5\) and \(\mu_1=1\) for \(\xi_\parallel\). In the most general case, we have \(N\) wedges \(\xi_{w,i}\), with \(\mu_0 = (i-1)/N\) and \(\mu_1=i/N\), for \(1\le i \le N\).

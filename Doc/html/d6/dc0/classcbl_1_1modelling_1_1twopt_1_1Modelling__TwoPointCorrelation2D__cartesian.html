)

set the dispersion model to fit the 2D two-point correlation function, in Cartesian coordinates

the 2D two-point correlation function, in Cartesian coordinates, is computed by cbl::modelling::twopt::xi2D_dispersion

the model has 5 parameters:

  • \(f(z)\sigma_8(z)\)
  • \(b(z)\sigma_8(z)\)
  • \(\sigma_v\)
  • \(\alpha_\perp=\frac{D_{\rm A,1}(z)}{D_{\rm A,2}(z)}\)
  • \(\alpha_\parallel=\frac{H_2(z)}{H_1(z)}\)
Parameters
fsigma8_priorprior for the parameter \(f(z)\sigma_8(z)\)
bsigma8_priorprior for the parameter \(b(z)\sigma_8(z)\)
sigmav_priorprior for the parameter \(\sigma_v(z)\)
alpha_perp_priorprior for the parameter \(\alpha_\perp=\frac{D_{\rm A,1}(z)}{D_{\rm A,2}(z)}\); a Dirac delta function centred on 1, by default
alpha_par_priorprior for the parameter \(\alpha_\parallel=\frac{H_2(z)}{H_1(z)}\); a Dirac delta function centred on 1, by default
